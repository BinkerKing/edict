# 碳循环减脂计划 - 用户数据分析报告

## 用户基础信息
- 体重：86公斤
- 体脂率：23%
- 瘦体重：66.2公斤 (86 × (1 - 0.23))
- 目标：减脂同时保持肌肉量

## 基础代谢率估算
- BMR ≈ 1,850-1,900千卡/天 (使用Mifflin-St Jeor方程)
- TDEE ≈ 2,200-2,500千卡/天 (根据活动水平)

## 减脂目标设定
- 每周减重目标：0.5-0.7公斤
- 每日热量缺口：350-500千卡
- 低碳日热量：1,700-2,000千卡
- 高碳日热量：2,300-2,700千卡

## 碳水化合物分配
- 低碳日：50-100克碳水 (约200-400千卡)
- 中碳日：150-200克碳水 (约600-800千卡)
- 高碳日：300-400克碳水 (约1,200-1,600千卡)

## 蛋白质和脂肪需求
- 蛋白质：每日155-172克 (体重×1.8-2.0g/kg)
- 脂肪：低碳日50-70克，高碳日40-60克

## 预期效果分析
- 预计4-6周可减少体脂2-4%
- 预计减重2-4公斤
- 肌肉量保持稳定或轻微增加
- 代谢适应风险较低
