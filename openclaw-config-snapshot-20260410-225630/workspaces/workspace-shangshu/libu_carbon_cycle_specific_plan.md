# 碳循环减脂计划 - 具体执行方案

## 用户信息
- 体重：86公斤
- 体脂率：23%
- 性别：男性（假设）
- 运动经验：中等

## 4周执行计划

### 第1-2周：适应期
- 高碳日：周六（力量训练日）
- 中碳日：周二、周四（力量训练日）
- 低碳日：周一、三、五、日（休息/轻度有氧）

### 第3-4周：强化期
- 高碳日：周三、周六（力量训练日）
- 中碳日：周二、周四（力量训练日）
- 低碳日：周一、五、日（休息/轻度有氧）

## 每日营养安排

### 低碳日（周一、三、五、日）
- 早餐：蛋白质30g + 蔬菜 + 少量坚果（约300卡）
- 午餐：蛋白质40g + 大量蔬菜 + 坚果/橄榄油（约400卡）
- 晚餐：蛋白质40g + 蔬菜 + 少量肉类脂肪（约400卡）
- 加餐：蛋白质粉或酸奶（约200卡）
- 总计：约1,300-1,500卡，碳水50-80g

### 中碳日（周二、周四）
- 早餐：蛋白质30g + 燕麦片30g + 少量水果（约400卡）
- 午餐：蛋白质40g + 蔬菜 + 红薯100g（约500卡）
- 晚餐：蛋白质40g + 蔬菜 + 糙米50g（约450卡）
- 训练前后：香蕉半根或能量棒（约200卡）
- 加餐：蛋白质粉（约150卡）
- 总计：约1,700-1,900卡，碳水150-200g

### 高碳日（周六）
- 早餐：蛋白质30g + 燕麦片50g + 水果（约500卡）
- 上午加餐：香蕉 + 能量棒（约250卡）
- 午餐：蛋白质40g + 蔬菜 + 米饭100g + 土豆（约600卡）
- 训练前：碳水补充（约200卡）
- 训练后：蛋白质粉 + 高GI碳水（约300卡）
- 晚餐：蛋白质40g + 蔬菜 + 意大利面75g（约550卡）
- 加餐：水果或甜点（约200卡）
- 总计：约2,400-2,600卡，碳水300-400g

## 训练安排

### 力量训练日（周二、周四、周六）
- 热身：10分钟轻度有氧
- 主训练：60-75分钟复合动作
- 有氧：20分钟中等强度（非高碳日）
- 拉伸：10分钟

### 休息日（周一、三、五、日）
- 轻度有氧：30-45分钟步行或骑行
- 或瑜伽/拉伸：45分钟

## 补剂建议
- 蛋白粉：每日1-2次，每次25-30g
- 鱼油：每日2-3g
- 维生素D：每日2000-4000IU
- 钾镁：低碳日补充

## 进度跟踪
- 每周固定时间称重
- 每两周测量体成分
- 记录训练表现
- 监控能量水平和恢复情况
