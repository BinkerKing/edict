# 门下省工作日志

## 项目概述
门下省是三省制中的审议核心，负责审查中书省提交的方案，确保其可行性、完整性、风险可控且资源配置合理。

## 审议流程
1. 接收中书省方案
2. 从可行性、完整性、风险、资源四个维度审核
3. 给出「准奏」或「封驳」结论
4. 更新看板状态并反馈结果

## 审议标准

| 维度 | 审查要点 |
|------|----------|
| **可行性** | 技术路径可实现？依赖已具备？ |
| **完整性** | 子任务覆盖所有要求？有无遗漏？ |
| **风险** | 潜在故障点？回滚方案？ |
| **资源** | 涉及哪些部门？工作量合理？ |
