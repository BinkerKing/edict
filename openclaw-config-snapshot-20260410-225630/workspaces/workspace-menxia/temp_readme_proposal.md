# 三省六部项目 - README模板

## 项目概述
三省六部是中国古代中央政府的行政体系，包括中书省、门下省、尚书省三省，以及吏部、户部、礼部、兵部、刑部、工部六部。

## 架构说明
### 三省职能
- **中书省**：负责决策制定与政策起草
- **门下省**：负责审议把关与方案审查
- **尚书省**：负责执行落实与事务管理

### 六部职能（预留）
- 吏部：人事管理
- 户部：财政管理
- 礼部：礼仪制度
- 兵部：军事事务
- 刑部：司法刑狱
- 工部：工程营造

## 使用指南
### 任务流转
1. 中书省提出方案
2. 门下省审议把关
3. 尚书省执行落实
4. 完成后归档记录

### 状态管理
- 使用看板系统跟踪任务进度
- 按规定格式更新任务状态

## 贡献指南
- 提交方案时需遵循工作协议
- 按规定格式填写任务ID和内容
- 及时更新任务状态

## 许可证
遵循中国古代政治制度传统

- 尚书省：负责执行管理
