# 三省六部项目

本项目模拟中国古代最著名的政治制度——三省六部制，这是一个高度系统化的政府组织架构，体现了古代中国在国家治理方面的卓越智慧。

## 项目概述

三省六部制是中国古代封建社会一套中央官制，确立于隋朝，完善于唐朝，此后一直到清末六部制被推翻为止。这套制度将国家政务分为决策、审议、执行三个环节，形成了相互配合又相互制约的权力运行机制。

## 三省架构

### 中书省（决策机构）
- **职能**：负责国家政策的起草与制定，相当于现代政府的政策研究室
- **长官**：中书令（正二品），副职为中书侍郎
- **主要职责**：秉承君主旨意，草拟诏令，参与国家重大决策的制定

### 门下省（审议机构）
- **职能**：负责审核中书省起草的诏令，具有封驳权
- **长官**：侍中（正二品），副职为门下侍郎
- **主要职责**：审议中书省所拟诏敕，如认为不当可予以驳回，确保政策的合理性和可行性

### 尚书省（执行机构）
- **职能**：负责执行国家政令，统领六部
- **长官**：尚书令（正二品），但唐太宗曾任尚书令，故其后不置，以左右仆射代行其职
- **主要职责**：承宣政令，统领六部九卿等执行机构，是国家行政的核心枢纽

## 六部架构

### 吏部（人事部门）
- **职能**：负责全国文官的任免、考核、升降、调动等事务
- **长官**：吏部尚书（正三品）
- **下属四司**：吏部司、司封司、司勋司、考功司

### 户部（财政部门）
- **职能**：管理全国土地、户籍、赋税、财政收支等
- **长官**：户部尚书（正三品）
- **下属四司**：户部司、度支司、金部司、仓部司

### 礼部（教育文化外交部门）
- **职能**：主管国家祭祀、礼仪、科举考试及外交事务
- **长官**：礼部尚书（正三品）
- **下属四司**：礼部司、祠部司、膳部司、主客司

### 兵部（军事部门）
- **职能**：负责武官选用、军队训练、武器装备管理等军务
- **长官**：兵部尚书（正三品）
- **下属四司**：兵部司、职方司、驾部司、库部司

### 刑部（司法部门）
- **职能**：主管全国司法审判事务，复核刑狱案件
- **长官**：刑部尚书（正三品）
- **下属四司**：刑部司、都官司、比部司、司门司

### 工部（建设部门）
- **职能**：管理国家工程建设、水利、交通、工匠等事务
- **长官**：工部尚书（正三品）
- **下属四司**：工部司、屯田司、虞部司、水部司

## 运行机制

### 决策流程
1. **中书出令**：中书省根据皇帝旨意或国家需要，起草诏令
2. **门下审令**：门下省对诏令进行审核，如有异议可封驳
3. **尚书行令**：经门下省审核通过后，交由尚书省执行

### 相互制约关系
- 中书省负责决策，但其决策需经门下省审核
- 门下省有封驳权，但不能自行决策
- 尚书省负责执行，但无决策权
- 三省相互配合又相互制约，有效防止个人专断

## 历史意义

三省六部制是中国古代政治制度的重大创新，实现了决策、审议、执行的相对分离，提高了行政效率，加强了中央集权，在中国政治制度史上具有重要地位，对后世影响深远。

## 项目特点

1. **权力制衡**：通过三省之间的相互制约，防止个人专权
2. **职责明确**：各部门职能清晰，避免重复劳动
3. **流程规范**：政令从起草到执行有明确的程序
4. **效率提升**：分工合作，提高行政效率
