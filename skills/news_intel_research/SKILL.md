# news_intel_research

用于“情报处/前沿早报”场景。目标是拿到**真实可访问**、**中文可读**、**来源可信**的新闻条目。

## 触发条件

- 用户要求：立即采集、前沿早报、热点新闻、按分类要闻
- 任务要求：每分类 5 条以上、需要来源链接、需要中文摘要

## 输入

- 新闻分类列表（如：政治、军事、经济、AI大模型）
- 每分类目标条数（通常 5-8）
- 时间要求（默认优先最近 24-72 小时）

## 处理流程

1. 先按分类做中文关键词检索，优先查找“最新/突发/热议”新闻。
2. 只选公开可访问的文章详情页，不要列表页、搜索页、404 页、跳转空壳页。
3. 优先采用知名媒体来源（如新华社、人民网、央视新闻、财新、澎湃、第一财经、界面、36氪、华尔街见闻等）。
4. 生成中文标题与中文摘要（可改写，但不歪曲原意）。
5. 输出结构化 JSON，保留原文链接与来源网站名。

## 输出规范

只输出 JSON：

```json
{
  "categories": {
    "经济": [
      {
        "title": "中文标题",
        "summary": "中文摘要",
        "link": "https://...",
        "source": "财新网",
        "pub_date": "2026-04-13 20:30"
      }
    ]
  }
}
```

## 注意事项

- 不要伪造链接；链接必须可打开。
- 不要输出英文标题/英文摘要。
- 同一分类内避免重复报道（同链接或同事件重复）。
- 若某分类不足目标条数，要继续检索补齐，不要直接放弃。
